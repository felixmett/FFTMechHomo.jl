# FFTMechHomo.jl

A Julia implementation of FFT-based solvers for micromechanical boundary value problems in heterogeneous materials.

## Installation

This project is not registered in the Julia General Registry. It can be installed directly from the source repository:

```julia
julia> ]add https://github.com/felixmett/FFTMechHomo.jl
julia> using FFTMechHomo
```
